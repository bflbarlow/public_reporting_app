# Seeker Universe Hologram

A force-directed galaxy visualization of seekers, referrals, programs, and engagement with full interactive controls.

## Features

- **Interactive force-directed graph** of seekers and programs
- **Zoom & Pan**: Mouse wheel to zoom, drag background to pan
- **Node Dragging**: Click and drag any seeker node to reposition it
- **Physics Controls**: Pause/resume physics simulation
- **Smart View Controls**: Reset zoom or auto-center view
- **Touch Support**: Pinch to zoom, drag to pan on mobile devices
- **Keyboard Shortcuts**: Space (pause/resume), R (reset), C (center)
- Seekers are nodes, programs are gravity wells
- Edge colors represent referral status
- Click/tap on any seeker node to view details
- Animated physics simulation with adjustable parameters

## Data Sources

- Seeker profiles
- Referrals
- Programs
- Referral status history
- Goals, notes, and user activity counts

## Parameters

This report has no parameters and displays all data.

## Usage

Access via `/api/embed?report_id=seeker_universe` (requires HMAC signature) or use the URL generated by the reporting app.

## Interactive Controls

### Mouse/Trackpad
- **Zoom**: Scroll wheel
- **Pan**: Click and drag empty space
- **Drag Nodes**: Click and drag any seeker node
- **Select Node**: Click (without dragging) to show details

### Touch Devices
- **Zoom**: Pinch with two fingers
- **Pan**: Drag with one finger
- **Drag Nodes**: Touch and hold a node, then drag
- **Select Node**: Quick tap on a node

### Keyboard Shortcuts
- **Space**: Toggle physics simulation
- **R**: Reset view to default zoom/pan
- **C**: Auto-center view on all nodes

### On-Screen Controls
- **Reset View**: Returns to default zoom/pan
- **Center**: Auto-centers view on all nodes
- **Pause/Resume Physics**: Toggles the physics simulation

## Development Notes

- Uses HTML5 Canvas for rendering with transformation matrix for zoom/pan
- Physics constants (FRICTION, SPRING, GRAVITY) can be adjusted for different visual effects
- Node brightness scales with total engagement (goals + notes + activity)
- Edge colors:
  - Completed: green
  - In Progress: yellow
  - Cancelled: red
  - Other/unknown: blue
- View transformation maintains visual clarity at all zoom levels
